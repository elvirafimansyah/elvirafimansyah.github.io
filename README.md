### Running Locally
1. Clone the repo
   ```sh
   git clone https://github.com/elvirafimansyah/elvirafimansyah.github.io.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
2. ```sh
   npm run start
   ```
3. Navigate to http://localhost:3000.
